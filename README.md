**Will it keep working after Roblox updates?**  
Yes. The auto-update engine watches for Roblox changes and updates the executor as needed, so manual patching is not required.

**What makes Synapse different from other executors?**  
Synapse combines an in-app script hub, a very small footprint, and a built-in debugger, which makes it practical for both newcomers and experienced users.

**Can Synapse get my Roblox account banned?**  
Any script executor use goes against Roblox's Terms of Service, so account action is always possible. Use caution, and preferably test on alternate accounts.

**Where are scripts and settings kept?**  
All local data is stored in the `Synapse/config/` and `Synapse/scripts/` folders. There is no telemetry or cloud storage involved.
